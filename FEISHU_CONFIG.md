## 🚀 配置飞书独立通知（5 分钟）

### 第 1 步：获取飞书 webhook URL

#### 方法 A：飞书群机器人

1. **打开飞书**
2. **创建一个群**（或选择现有群）
   - 可以只拉自己一个人的群
3. **群设置 → 群机器人**
4. **添加机器人 → 自定义机器人**
5. **设置名称：** 健康管家
6. **复制 webhook URL**
   - 格式：`https://open.feishu.cn/open-apis/bot/v2/hook/xxx`

#### 方法 B：飞书单聊机器人

1. **打开飞书**
2. **搜索"飞书助手"**
3. **添加自定义机器人**
4. **复制 webhook URL**

---

### 第 2 步：配置到健康管理系统

**编辑配置文件：**
```bash
# 文件位置：C:\Users\Administrator\.health_guardian_config.json
```

**添加飞书 webhook：**
```json
{
  "notifications": {
    "feishu_webhook": "https://open.feishu.cn/open-apis/bot/v2/hook/你的 webhook URL",
    "enabled": true,
    "channel": "feishu"
  }
}
```

**或者告诉我 webhook URL，我帮你配置！**

---

### 第 3 步：设置定时任务

**运行命令：**
```powershell
powershell -ExecutionPolicy Bypass -File "github\davidme6\health-guardian\scripts\setup_feishu_tasks.ps1"
```

**或者手动设置：**
1. 打开"任务计划程序"
2. 创建基本任务
3. 名称：HealthGuardian-Feishu
4. 触发器：每天，重复间隔 10 分钟
5. 操作：启动程序
   - 程序：`python`
   - 参数：`github\davidme6\health-guardian\scripts\feishu_notify.py`

---

### 第 4 步：测试

**发送测试消息：**
```bash
python github\davidme6\health-guardian\scripts\feishu_notify.py test
```

**如果成功，你会在飞书收到：**
```
【健康管理系统测试】

如果你收到这条消息，说明飞书通知配置成功！✅
```

## ⚠️ 注意事项

1. **webhook URL 保密**
   - 不要分享到公开平台
   - 相当于密码

2. **网络要求**
   - 需要能访问飞书服务器
   - 公司网络可能需要代理

3. **通知频率**
   - 每 10 分钟检查一次
   - 只在有提醒时发送
   - 不会频繁打扰

4. **免打扰时间**
   - 22:00 - 07:00 不发送
   - 避免影响休息
