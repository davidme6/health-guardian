# 健康管理系统 - OpenClaw 集成说明

## 问题诊断

**问题：** 健康管理系统没有主动发送通知到飞书

**原因：** 
1. Windows 任务计划程序的提醒脚本写入文件，但 OpenClaw 没有自动读取
2. 需要 OpenClaw 主动检查并发送消息

---

## 解决方案

### 方案 1：通过 OpenClaw 心跳检查（推荐）

在 `HEARTBEAT.md` 中添加定期检查任务，让 OpenClaw 每小时检查健康提醒。

**配置：**
```markdown
# HEARTBEAT.md

## 健康提醒检查（每小时）

检查命令：
```
python github\davidme6\health-guardian\scripts\notify.py
```

如果有提醒消息，发送到飞书。
```

### 方案 2：直接对我说（最简单）

直接对我说：
- "查看健康提醒"
- "今天应该做什么"
- "健康检查"

我会立即检查并告诉你！

### 方案 3：定时 Cron 任务

设置 OpenClaw Cron 任务，在指定时间自动触发提醒。
