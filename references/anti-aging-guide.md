## 💎 三大黄金标准

### 1️⃣ 好好睡觉

**为什么重要：**
- 夜间是细胞修复黄金时间
- 生长激素分泌高峰
- 大脑清理代谢废物
- 免疫系统恢复

**实践建议：**
```
✅ 保证 7-8 小时睡眠
✅ 22:30 前入睡
✅ 遵循自然作息（日出而作，日入而息）
✅ 睡前 1 小时关闭电子设备
✅ 保持卧室黑暗、安静、凉爽
✅ 睡眠呼吸暂停患者使用呼吸机
```

### 2️⃣ 好好运动

**为什么重要：**
- 促进新陈代谢
- 增强心肺功能
- 维持肌肉量
- 减少慢性炎症

**实践建议：**
```
✅ 每周 150 分钟中等强度运动
✅ 有氧 + 力量训练结合
✅ "动对"比"多动"更重要
✅ 选择适合自己的运动
✅ 持之以恒，不要过度
✅ 运动后充分休息
```

**推荐运动：**
- 🏃 快走或慢跑（30 分钟）
- 🏊 游泳（全身运动）
- 🚴 骑行（心肺锻炼）
- 💪 俯卧撑、深蹲（力量训练）
- 🧘 瑜伽、拉伸（柔韧性）

### 3️⃣ 好好吃饭

**为什么重要：**
- 营养是细胞修复原料
- 饮食影响炎症水平
- 肠道健康影响全身

**实践建议：**
```
✅ 增加豆类摄入（豆腐、豆浆）
✅ 多吃全谷物（糙米、燕麦、全麦）
✅ 适量坚果（核桃、杏仁）
✅ 大量蔬菜水果
✅ 深海鱼（Omega-3）

❌ 减少红肉（猪牛羊）
❌ 避免加工肉类（香肠、培根）
❌ 少吃精制碳水（白米饭、白面包）
❌ 少油少盐
❌ 避免过饱（七分饱）
```

**清茶淡饭实践：**
- 🍵 每天喝绿茶（茶多酚抗氧化）
- 🥬 多吃蔬菜（纤维、维生素）
- 🍚 主食粗细搭配
- 🐟 优质蛋白（鱼、豆、蛋）
- 🥜 健康脂肪（坚果、橄榄油）

## 🧘 心理调节（不劳心力）

### 压力对衰老的影响
- 加速端粒缩短
- 增加皮质醇（分解肌肉）
- 影响睡眠质量
- 导致慢性炎症

### 减压方法
```
🧘 冥想 - 每天 10 分钟
🌬️ 深呼吸 - 4-7-8 呼吸法
🚶 散步 - 自然环境最佳
📝 写日记 - 记录感恩事项
🎵 听音乐 - 舒缓放松
😴 充足睡眠 - 最好的恢复
```
