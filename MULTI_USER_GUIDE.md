## 👥 添加新成员

### 方式 1：告诉我（推荐）

**直接告诉我新成员信息：**

> "添加新用户：张三，飞书 webhook 是：https://..."

**然后我会问：**
1. 身高、体重、目标体重
2. 年龄、性别

6. 睡觉时间

**我帮你配置好一切！**

```bash
python multi_user_notify.py add "张三" "https://飞书 webhook URL"
```

**然后更新健康档案：**
编辑 `C:\Users\Administrator\.health_users.json`

## 🎯 每个人的专属数据

### 系统会记录：

| 数据类型 | 说明 |
|----------|------|
| **基本信息** | 身高、体重、年龄、性别 |
| **健康目标** | 目标体重、每日饮水、运动目标 |
| **健康状况** | 疾病史、使用设备（如呼吸机） |
| **偏好设置** | 通知风格、免打扰时间 |
| **进度统计** | 今日饮水、运动、最新体重 |

### 通知会根据：

- ✅ 姓名个性化称呼
- ✅ 当前体重和目标体重
- ✅ 今日已完成进度
- ✅ 健康状况（特殊提醒）
- ✅ 个人目标设定
