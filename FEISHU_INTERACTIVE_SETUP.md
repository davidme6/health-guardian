# 📱 飞书双向互动配置指南

**让健康管理系统能够接收并回复你的消息！**

---

## ❌ 当前问题

**现状：**
- ✅ 系统可以发送通知到飞书群
- ❌ 系统无法接收你在群里的回复
- ❌ 无法实现互动对话

**原因：**
- Webhook 是单向的（只能发送）
- 需要配置飞书开放平台事件订阅

---

## ✅ 解决方案

### 方案 A：配置飞书事件订阅（推荐）⭐⭐⭐⭐⭐

**步骤：**

#### 1. 创建飞书应用

1. 访问：https://open.feishu.cn/app
2. 点击"创建企业自建应用"
3. 填写应用信息：
   - 应用名称：健康管家
   - 图标：随便选
4. 点击"创建"

#### 2. 配置机器人

1. 进入应用 → 添加机器人
2. 填写机器人信息：
   - 名称：健康管家
   - 描述：你的专属健康管理助手
3. 复制：
   - App ID
   - App Secret

#### 3. 配置事件订阅

1. 进入应用 → 事件订阅
2. 开启事件订阅
3. 配置订阅地址：
   ```
   http://你的服务器地址:端口/feishu/event
   ```
4. 订阅事件：
   - ✅ 接收消息
   - ✅ 群聊消息
   - ✅ @机器人消息

#### 4. 配置机器人加入群

1. 进入应用 → 机器人
2. 点击"添加到群聊"
3. 选择"健康管理通知"群
4. 确认添加

#### 5. 配置验证 Token

1. 事件订阅页面
2. 复制 Verification Token
3. 保存到配置文件
