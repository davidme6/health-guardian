## 🚀 配置步骤（5 分钟）

### 第 1 步：创建飞书通知群

1. **打开飞书**
2. **创建群聊**
   - 点击右上角「+」→ 「创建群聊」
   - 群名称：`健康管理通知`（或其他你喜欢的名字）
   - **可以只拉自己一个人**（这样就不会打扰别人）

3. **进入群设置**
   - 点击右上角群设置图标
   - 找到「群机器人」

---

### 第 2 步：添加机器人

1. **点击「添加机器人」**
2. **选择「自定义机器人」**
3. **设置机器人信息：**
   - 名称：`健康管家`
   - 头像：随便选一个
4. **复制 webhook URL**
   - 格式：`https://open.feishu.cn/open-apis/bot/v2/hook/xxxxx`
   - **这个 URL 很重要，一定要复制！**

---

**方式 B：手动配置**

编辑配置文件：
```bash
# 文件位置：C:\Users\Administrator\.health_guardian_config.json
```

添加飞书 webhook：
```json
{
  "notifications": {
    "feishu_webhook": "https://open.feishu.cn/open-apis/bot/v2/hook/你的 webhook URL",
    "enabled": true,
    "channel": "feishu"
  }
}
```
